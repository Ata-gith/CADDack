# CADDack
CADDack — An integrated toolkit and pipeline repository for Computational Aided Drug Design (CADD) featuring classical analyses, molecular docking, QSAR modeling, and Graph Neural Network (GNN) based molecular property prediction.


Welcome to **caddack**, a comprehensive repository dedicated to tools, scripts, and workflows for Computational Aided Drug Design (CADD). This project integrates classical cheminformatics, molecular docking, QSAR modeling, and modern Graph Neural Network (GNN) approaches for molecular property prediction and drug discovery.

---

## Features

- **Molecular Docking**  
  Automated workflows for docking ligands to target proteins using tools like AutoDock Vina and EasyDock.

- **QSAR Modeling**  
  Data processing pipelines for building classical QSAR models using molecular descriptors and fingerprints.

- **Graph Neural Networks (GNNs)**  
  Implementations and examples using DGL-LifeSci and PyTorch for GNN-based molecular property prediction.

- **Data Integration**  
  Scripts to query and extract bioactivity data from public databases like ChEMBL.

- **Visualization & Analysis**  
  Utilities for interaction analysis (e.g., hydrogen bonds, binding site metrics) and result visualization.
